### Q: Diagnose a system that passes benign safety tests but fails under retrieved or multimodal adversarial content.
* **Difficulty:** Principal
* **Category:** Debugging
* **The 10-Second Pitch:** Reproduce the exact content path and find where trust/provenance or policy enforcement disappeared. Benign tests likely exercised direct user input but not indirect multimodal/tool channels.
* **The Deep Dive:** Preserve a replay bundle containing raw source bytes, rendered/OCR text, parser version, chunk IDs, embedding/index/reranker versions, retrieved order, final serialized context, model/sampling parameters, tool schemas/results, policy decisions, and identity. Reproduce with side effects disabled. Delta-debug by removing modalities, chunks, metadata, and instructions; move the same payload between user input, OCR, retrieved text, tool output, and memory to identify the unprotected channel. Inspect invisible HTML, alt text, Unicode controls, image layers, audio steganography/transcription, parser differentials, and truncation that may remove the trusted instruction while retaining the payload.

  Follow the causal chain past model output. Did the model merely repeat content, propose an unsafe call, or did an executor actually authorize it? If a harmful effect occurred, determine why principal/resource authorization, argument validation, approval binding, egress control, or capability scope accepted the proposal. Fix the missing boundary first; prompt tuning and classifiers are defense-in-depth. Convert the minimized exploit into tests at ingestion, retrieval, context assembly, model proposal, policy enforcement, and executor levels, plus transformations that preserve intent but change encoding/language/layout.
* **Production Reality & Tradeoffs:** Malicious content persists in derived chunks, vector indexes, caches, summaries, and long-term memory after the source is removed. Quarantine by provenance, purge derivatives, invalidate affected caches, and rotate any exposed credential. Replay against both previous and fixed versions, validate benign utility, and test attacker adaptations. Aggregate pass rate is inadequate—report effect severity, channel, privilege reached, and whether deterministic controls contained the model failure.
* **Red Flag:** Responding only by adding the malicious phrase to a blocklist.
